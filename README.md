# SafraCafé

- Este aplicativo mobile está sendo desenvolvido para auxiliar produtores de café na gestão da colheita, permitindo acompanhar produtividade, gerenciar funcionários e prever futuras safras de forma simples e organizada.
- O objetivo é otimizar o controle da colheita, reduzir erros manuais e apoiar a tomada de decisão com informações claras e acessíveis.
- O Software está sendo desenvolvido na linguagem Kotlin+Compose e SQLite para permanência dos dados.
